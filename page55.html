Электронное учебное пособие создано в формате HTML с помощью бесплатного программного приложения TurboSite на основе оригинального учебного пособия: Вощукова Е.А. Физические основы строительной акустики: учебное пособие для студентов очного и заочного обучения (направление подготовки бакалавров "Строительство")/ Е.А. Вощукова. - Брянск: БГИТА, 2011. - 96 с.

 

Минимальные системные требования:

операционная система — Windows 7/Vista/XP/2000;

процессор — Pentium 4 с частотой 1,5 ГГц, либо Athlon XP 1500+ и выше; оперативная память не менее 512 Мб;

не менее 500 Мб свободного места на жестком диске;

наличие звуковой карты.

 

На компьютере должно быть установлено следующее программное обеспечение: один из веб-браузеров (Mozilla Firefox, Internet Explorer, GoogleChrome); Adobe FlashPlayer 9.0 и выше; программа для чтения файлов в формате pdf (Adobe Acrobat Reader); программа для просмотра презентаций (Windows PowerPoint). 

 

Для просмотра видеофрагментов и прослушивания аудиофайлов (формат mp4) используется проигрыватель Longtail, который загружается вместе с электронным учебным пособием.
